## Installation
1. Clone the repository:
```yaml
git clone https://github.com/RichardSouzza/TicTacToe
```
2. Access the project folder:
```yaml
cd TicTacToe
```
3. Install the dependencies:
```yaml
pip install -r requirements.txt
```
4. Run the game:
```yaml
python main.py
```
